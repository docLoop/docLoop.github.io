Makes sure there is an issue corresponding to the provided annotation id. If there is no such issue existing yet, then create a dummy issue and return its issue number. This is useful if a reply event fires before its parent annotation event. This can happen if the annotation is old and has never been imported or the import has been delayed for some reason.
